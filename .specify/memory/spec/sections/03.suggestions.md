### 13. Ek Öneriler / Zenginleştirmeler

1. **Boolean kısaltmaları:** `doğru`, `yanlış`
2. **Liste ve sözlük doğal dil fonksiyonları:**

```otağ
alis_listesi = liste!
listeye_ekle! liste: alis_listesi, değer: 42
telefon_defteri = sözlük!
sözlüğe_ekle! sözlük: telefon_defteri, anahtar: "Ali", değer: "555-1234"
```

3. **Zaman ve bekleme:**

```otağ
bekle! saniye: 2
şimdi!
```

4. **TDD odaklı yorumlar ve test fonksiyonları:**

```otağ
test! isim: "toplama testi"
    x = 5
    y = 10
    sonuç = topla! a: x, b: y
    söyle sonuç
```

5. **Opsiyonel parametre desteği:** `parametre: değer [varsayılan]`
