## Otağ Dili – Kapsamlı Gramer Taslağı (EBNF Tarzı)

### 1. Genel Notasyon

- `< >` → nonterminal
- `"..."` → terminal (literal)
- `[ ... ]` → opsiyonel
- `{ ... }` → tekrar edebilir
- `|` → alternatif

---

### 2. Program

```
<program> ::= { <statement> }
```
